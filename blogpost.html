Que es Git

Es un software de control de versiones diseñado por Linus Torvalds en el 2005, este también te permite llevar un control de versiones cuando estas trabajando en equipo con otros Desarrolladores, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

Hoy en día, Git es, con diferencia, el sistema de control de versiones moderno más utilizado del mundo.
